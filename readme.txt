Easily add the Priceline travel widget to your own website in just a few clicks.

== Description ==

This plugin creates a searchbox for Priceline Partner Network. You must already be an affiliate with PPN to be able to use this plugin. If you are not an affiliate, you may apply at http://www.pricelinepartnernetwork.com/privatelabel.

== Features ==

* Easy to use: Perform the initial setup and you're done!
* Customizable: Tons of customizations and theme support allow you to tailor your searchbox as you see fit.
* Support: Affiliate support and frequent updates keep you up to date.

== Installation ==

1. Download the zip package of the plugin.
2. Login to your WordPress admin section.
3. Click on "Plugins" and then click "Add New". Then click "Upload Plugin" and select the zip file and click "Install Now".
4. Back in the Plugins section, activate the Priceline Partner Network Official Searchbox plugin and then click on "Settings".
5. On the settings page, enter your refid and api key from the PPN Admin panel and click verify.
6. Once verified you are good to go! You can use the widget as is, or customize it under Appearance > Widgets.
